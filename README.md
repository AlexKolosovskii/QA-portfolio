# QA-portfolio
Привет!  
Я изучаю тестирование и применяю ключевые техники тест-дизайна на реалистичных сценариях.  
В этом репозитории — примеры моих решений, созданные с использованием:

- **Эквивалентного разбиения (EP)**
- **Анализа граничных значений (BVA)**
- **Попарного тестирования (Pairwise)**
- **Таблицы принятия решений (Decision Table)**
- **Диаграммы переходов состояний (State Transition)**

Каждый кейс оптимизирован для максимального покрытия при минимальном числе тестов.
